# Playfair Cipher Encryption and Decryption Application

## Introduction
The Playfair cipher is a manual symmetric encryption technique and was the first practical digraph substitution cipher. It was invented in 1854 by Charles Wheatstone and later promoted by Lord Playfair.

## Features
* **Key Square Generation:** Creates a 5x5 grid based on a secret key.
* **Digraph Encryption:** Encrypts pairs of letters instead of single characters.
* **Graphical User Interface:** Built using Python's Tkinter for ease of use.

## How to Run
1. Ensure Python is installed on your system.
2. Download the `main.py` file from this repository.
3. Run the file using: `python main.py`
